1. Introduction

In my day, I may have to work on several different project and keep track separately of how much time I spent working on each project. If you are like me, then Time Tracker is for you.

You can enter a list of tasks, start the clock, and it will count the number of minutes you spend on the highlighted task.

Time Tracker was inspired by Harald Tveit Alvestrand's very useful utility called titrax, the only failing of which is that it is based on the Xt toolkit. One day Time Track will do all that titrax does, but not today.
